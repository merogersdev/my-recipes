# My Recipes (Add Screenshot)

![Focus App Screenshot](https://my-portfolio-screens.s3.ca-central-1.amazonaws.com/focus/focus-screenshot-lg.png)

## Quick Summary

Full-Stack MERN (MongoDB, Express, React, Node) application to store recipes.

## Tech Stack

- React
- Webpack
- Typescript
- TailwindCSS
- Redux Toolkit
- Node.js & Express
- MongoDB & Mongoose
- Docker
- Github Actions for CI/CD
