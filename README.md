# My Recipes

## Quick Summary

Full-Stack MERN (MongoDB, Express, React, Node) application to store recipes.

## Tech Stack

- Vite or Next.js
- Typescript
- TailwindCSS
- Redux Toolkit
- Node.js & Express
- Docker
- Github Actions for CI/CD
